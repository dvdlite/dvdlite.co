#### Colophon
This website is artisanally hand-crafted primarily in Massachusetts, USA, using [Markdown](https://daringfireball.net/projects/markdown/), [Visual Studio Code](https://code.visualstudio.com), the [11ty](https://www.11ty.dev) static site generator, and the [eleventy-excellent v2.0](https://github.com/madrilene/eleventy-excellent) starter. The source code is hosted and publicly viewable on [GitHub](https://github.com/dvdlite/dvdlite.co) and deployed using the [Netlify](https://netlify.com) serverless platform.
